## Os 5 Padrões que Aparecem em TODAS as Vendas SMQ

1. **Personalização imediata** — sempre nome + interesse específico na abertura
2. **Educação financeira consultiva** — explicar FGTS, crédito e parcelas antes de vender
3. **Mídia estratégica** — áudios para humanizar, fotos/PDFs para tangibilizar
4. **Paciência ativa** — manter o cliente vivo sem pressão, com follow-up com motivo real
5. **Fechamento por facilitação** — remover barreiras em vez de pressionar

## Caso 1 — Glaucia x Guilherme (Supreme Vila Romana/Cury)
**Duração:** ~1 semana | **Resultado:** Venda fechada no lançamento

- Manteve engajamento mesmo com objeção de entrada

```
Lead: "O que pega é que não temos uma entrada boa."
Guilherme: "Consigo parcelar esse valor... e podemos ver formas de

```

*Urgência baseada em fato real + tom descontraído + incentivo concreto.*

## Caso 4 — Kaique x Guilherme (Raiz Home Clube/Direcional)
**Duração:** 1 dia | **Resultado:** Venda fechada (atendimento B2B — corretor parceiro)

### Diferenças no atendimento B2B vs cliente final

**Linguagem:**
- Mais técnica e direta (jargão imobiliário aceito)
- "tubulação de água na torre", "parcelamento decrescente", "bônus adimplência"
- Tom de parceria: "de corretor para corretor"

**Velocidade:**
- Muito mais rápido — corretor já entende o processo
- Negociação de condições mais direta

**O bônus surpresa:**
```
Guilherme: "Como parceiro da construtora, recebo premiação de 
R$10.000 por venda. Vou abrir mão desse valor para aumentar 
o desconto e reduzir suas parcelas."
```
*Resultado: parcelamento decrescente a partir de R$794,58. Fechamento no mesmo dia.*

### Como Guilherme escolheu a unidade para Kaique
- **Sol e iluminação:** explicou que unidades finais 3 e 4 recebem sol da tarde (mas ficam na frente do lazer — barulho)
- **Ruído:** Kaique tinha preocupação com avenida. Guilherme explicou que a rua não é movimentada e a acústica das janelas minimiza
- **Tubulação:** esclareceu que tubulação de água passa só na Torre 1, unidade escolhida não afetada
- **Vista e privacidade:** unidade de ponta, janela extra, área verde atrás (sem rua)

### Aprendizado B2B
Corretor-parceiro é um cliente diferente. Precisa de: linguagem técnica + argumento de viabilidade financeira para o cliente final + bônus que aumentam comissão efetiva + confiança de que você vai fechar com o cliente dele.

## Playbook Definitivo SMQ — Protocolo de 8 Etapas

| Etapa | Ação | Objetivo | Ferramentas/Técnicas |
|---|---|---|---|
| 1. Abertura | Responder rápido, agradecer, confirmar interesse | Engajamento inicial personalizado | Nome + produto clicado |
| 2. Qualificação | Renda, FGTS, região, perfil, decisor | Entender potencial + alinhar com portfólio | Perguntas abertas em blocos |
| 3. Apresentação | Fotos, plantas, vídeos, simulação | Construir valor e despertar desejo | Multimídia + dados técnicos |
| 4. Objeções | Escutar com empatia, oferecer alternativas | Superar barreiras mantendo motivação | Áudios + flexibilidade |
| 5. Follow-up | Lembretes, atualizações, novidades | Manter engajado sem pressão | Mensagens curtas + motivo real |
| 6. Negociação | Bônus, parcelamento, descontos | Viabilizar o fechamento | Bônus surpresa + transparência |
| 7. Fechamento | Benefícios finais, documentação, próximos passos | Garantir compromisso | Plano financeiro claro |
| 8. Pós-Venda | Acompanhar aprovação, tirar dúvidas | Fidelização + indicações | Comunicação contínua |

## Biblioteca definitiva de mensagens vencedoras SMQ

### Abertura
```
"Bom dia, [Nome]! Tudo bem? Vi que se interessou pelo nosso 
lançamento em [local], isso mesmo?"

"Fala, [Nome]! Aqui é o [Corretor]. Está buscando imóvel na 
região? Temos algumas opções que podem te interessar."
```

### Qualificação
```
"Qual é a faixa de renda de vocês e até quanto pretendem 
investir? Assim consigo trazer opções alinhadas ao seu perfil."

"Conta pra mim um pouco do que vocês buscam no imóvel, para 
recomendar algo que realmente encaixe."
```

### Construção de valor
```
"Esse projeto tem uma localização excelente, perto de [ponto 
de referência], e entrega prevista para [data]."

"Neste mês, temos documentação 100% gratuita, o que representa 
uma economia de aproximadamente R$12 mil."
```

### Objeção: parcela alta
```
"Entendo sua preocupação com o valor da parcela. Podemos ajustar 
o parcelamento para parcelas decrescentes, facilitando o 
pagamento conforme a obra avança."
```

### Follow-up
```
"Só passando para lembrar da visita agendada amanhã, vou estar 
esperando vocês na recepção."

"Temos uma unidade bloqueada para vocês, mas é importante 
confirmar logo para garantir."
```

### Fechamento
```
"Como parceiro da construtora, posso abrir mão da minha 
premiação para aumentar o desconto e reduzir suas parcelas."

"Se fecharmos agora, consigo garantir documentação grátis 
e condições especiais de pagamento."
```

### Pós-venda
```
"Parabéns pela escolha, [Nome]! Vou acompanhar pessoalmente 
o andamento da sua aprovação. Qualquer dúvida, estou aqui."
```

## Comparativo dos estilos de atendimento SMQ

| Critério | Guilherme Nunes | Andrew |
|---|---|---|
| Tom | Próximo, descontraído, pessoal | Formal, estruturado, didático |
| Uso de áudio | Intenso — humaniza | Menor — compensa com docs |
| Ponto forte | Conexão emocional + paciência | Autoridade técnica + organização |
| Melhor para | Leads que precisam de confiança pessoal | Leads que precisam de clareza processual |
| Comum em ambos | Personalização, educação financeira, follow-up com motivo |

**Insight:** nenhum estilo é superior. Adaptar ao perfil do cliente é o diferencial.

## O que ensinar a novos corretores da SMQ

*(Baseado nos padrões dos corretores que fecharam)*

1. Nunca esqueça o nome do cliente — use sempre
2. Explique o porquê das perguntas — cria segurança
3. Envie conteúdo visual logo nos primeiros contatos — tangibiliza
4. Se o lead sumiu, volte com um motivo real — não "oi, tudo bem?"
5. A venda pode demorar 2 meses — não desista no 2º follow-up
6. Dominar o processo da Caixa é vantagem competitiva
7. Flexibilidade financeira (parcelamento, bônus, isenção) fecha vendas
8. Transparência na adversidade é o que cria lealdade
9. Gerencie as expectativas antes de criar — menos frustração, mais confiança
10. O pós-venda proativo gera indicações — e indicação é o lead mais barato
